Formulario Algebra

Álgebra

Exponentes y Radicales

am · an = am+n
(am)n = am·n
(a·b)n = an · bn
a-n = 1/an (a ≠ 0)
√(a·b) = √a · √b
√(a/b) = √a / √b (b ≠ 0)

Logaritmos

loga(x·y) = logax + logay
loga(x/y) = logax - logay
loga(xn) = n·logax
logab = logcb / logca (Cambio de base)

Factorización

a² - b² = (a + b)(a - b)
a³ ± b³ = (a ± b)(a² ∓ ab + b²)
(a ± b)² = a² ± 2ab + b²
(a ± b)³ = a³ ± 3a²b + 3ab² ± b³

Ecuaciones Cuadráticas

ax² + bx + c = 0 → x = [-b ± √(b² - 4ac)] / 2a
Discriminante: D = b² - 4ac
Si D > 0: 2 raíces reales distintas
Si D = 0: 1 raíz real doble
Si D < 0: 2 raíces complejas conjugadas

Trigonometría

Identidades Fundamentales

sen²θ + cos²θ = 1
1 + tan²θ = sec²θ
1 + cot²θ = csc²θ

Fórmulas de Suma y Resta

sen(α ± β) = senα cosβ ± cosα senβ
cos(α ± β) = cosα cosβ ∓ senα senβ
tan(α ± β) = (tanα ± tanβ) / (1 ∓ tanα tanβ)

Ángulos Dobles

sen(2θ) = 2 senθ cosθ
cos(2θ) = cos²θ - sen²θ = 2cos²θ - 1 = 1 - 2sen²θ
tan(2θ) = 2tanθ / (1 - tan²θ)

Ley de Senos y Cosenos

Ley de Senos: a/senA = b/senB = c/senC = 2R (R: radio circunferencia circunscrita)
Ley de Cosenos: c² = a² + b² - 2ab cosC

Transformaciones

senα + senβ = 2 sen[(α+β)/2] cos[(α-β)/2]
senα - senβ = 2 cos[(α+β)/2] sen[(α-β)/2]
cosα + cosβ = 2 cos[(α+β)/2] cos[(α-β)/2]
cosα - cosβ = -2 sen[(α+β)/2] sen[(α-β)/2]

Geometría Analítica

Distancia entre dos puntos

d = √[(x₂ - x₁)² + (y₂ - y₁)²]

Punto Medio

M = ((x₁ + x₂)/2, (y₁ + y₂)/2)

Ecuación de la Recta

Forma general: Ax + By + C = 0
Forma pendiente-ordenada: y = mx + b (m: pendiente, b: ordenada al origen)
Pendiente: m = (y₂ - y₁)/(x₂ - x₁)
Forma punto-pendiente: y - y₁ = m(x - x₁)
Rectas paralelas: m₁ = m₂
Rectas perpendiculares: m₁ · m₂ = -1

Secciones Cónicas

Circunferencia

(x - h)² + (y - k)² = r² (Centro en (h,k), radio r)

Elipse

(x - h)²/a² + (y - k)²/b² = 1 (Centro en (h,k), eje mayor 2a, eje menor 2b)
Excentricidad: e = c/a (c = √(a² - b²))

Hipérbola

(x - h)²/a² - (y - k)²/b² = 1 (Centro en (h,k), eje transversal horizontal)
(y - k)²/a² - (x - h)²/b² = 1 (Centro en (h,k), eje transversal vertical)
Asíntotas: y - k = ±(b/a)(x - h) (para la primera forma)
Excentricidad: e = c/a (c = √(a² + b²))

Parábola

(y - k)² = 4p(x - h) (Vértice en (h,k), eje horizontal, abre derecha si p>0)
(x - h)² = 4p(y - k) (Vértice en (h,k), eje vertical, abre arriba si p>0)

Coordenadas Polares

x = r cosθ, y = r senθ
r = √(x² + y²), θ = arctan(y/x)

Gráficas de Secciones Cónicas

Circunferencia

(x-h)² + (y-k)² = r²

Elipse

(x-h)²/a² + (y-k)²/b² = 1

Parábola (Vertical)

(x-h)² = 4p(y-k)

Hipérbola

(x-h)²/a² - (y-k)²/b² = 1
